En este componente formativo se profundizará sobre los modelos de virtualización a nivel del sistema operativo, adicionalmente se desarrollarán ejercicios que facilitarán el conocimiento a la plataforma Docker la cual es ampliamente utilizada por la industria de desarrollo de software.

Al finalizar este tema el aprendiz estará en capacidad de crear diversos aspectos en la plataforma Docker, además de construir, ejecutar, modificar y transformar contenedores Docker, a partir de imágenes locales, así como compartir una imagen local en el repositorio en la nube Docker Hub.
